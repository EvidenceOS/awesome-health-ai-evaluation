# Contributing to Awesome Health AI Evaluation

Thank you for contributing! This list curates evidence-based frameworks for evaluating health AI systems.

## How to Contribute

1. **Fork** this repository
2. **Add** your resource in the appropriate section
3. **Follow** the format: `- [Resource Name](URL) — Brief description`
4. **For structured data:** Add YAML/JSON entries to the `data/` directory following the existing schemas
5. **Submit** a pull request with a clear description

## Quality Criteria

- Resources must reference **peer-reviewed publications** or **established reporting standards**
- Include the **citation** (Author, Year, Journal) where applicable
- Study design resources should reference validated methodological frameworks
- Model cards and evidence chains should follow the templates in `data/templates/`

## Contribute & Publish

Top contributors are eligible for co-authorship on companion papers. See the README for details on our Human vs AI curation study.

## Code of Conduct

Be respectful, constructive, and focused on improving the resource for the community.
